## Sistema de Cadastro de Funcionários

Este programa permite gerenciar funcionários de uma empresa, com funções de cadastro, consulta e remoção.

## Funcionalidades
- Cadastro de novos funcionários com ID, nome, setor e salário

## Consulta de funcionários:
- Todos os funcionários cadastrados
- Por ID
- Por setor
- Remoção de funcionários pelo ID
- Menu interativo com validação de entradas

## Tecnologias
- Python 3

## Como executar
```bash
python sistema_funcionarios.py
